Online Voting- A Web Application


Purpose and Motivation

We have tried designing the application based onthe actual workings of the Election Commision of India and the way the actual elections are conducted. Below all the steps are described in detail to make the user understand how the website works.

Step 1: Register as a Voter

The first and foremost step in the voting procedure is to register yourself as a voter.

By the basic knowledge of how a voting and electoral system works, it is a prerequisite for any candidate to be first registered as a voter. So is for an Election officer. Thus, this is a necessary step towards using the election portal in any case.

REGISTERING AS A CANDIDATE

This is not a necessary step for everyone. Only those users need to register themselves as candidates, who wish to contest the elections. A candidate has to provide a photo, a scanned copy of authentic signature and the Constituency name that he wishes to contest elections from.

Upon registration as a candidate, a dashboard different than that of a voter will be visible.


Step 2: Login

After the successful registering of the voter the person is requires to login as Voter or Candidate as per the criterion satisfied. Then a dashboard appears that has options to cast a vote and to view results afterwards.








STEP3: CAST VOTE

Once you have logged in, and you are on your Dashboard, you can now view all your profile details and have an option to case the vote, clicking on which will lead you to a secure portal where you can cast the vote and later see the results also.

Features

Flask and MySQL

All the back-end has been developed by Flask and all the database has been maintained by MySQL Server.

Secured Data

We have secured all the data using inbuilt sha script so that no-one can decode the data. All the data lies in the encrypted format.

Convinient

The application i easy to use with enough features and is very comfortable.

Big Cost-Saver

The web-application saves the need for the poll booth and the cost is reduced to a great extent.
